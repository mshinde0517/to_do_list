# To-Do List Application (Python)

This project is a simple To-Do List Application developed using Python as part of my internship at CodSoft.

## 📖 Description
The application helps users manage and organize their daily tasks efficiently. Users can add, update, and track tasks easily.

## ⚙️ Features
- Add new tasks
- View all tasks
- Update tasks
- Mark tasks as completed

## 🛠️ Technologies Used
- Python

## 🚀 Learning Outcome
- Improved logic building
- Understanding of user input handling
- Practical implementation of Python concepts
